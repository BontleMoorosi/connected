# Connected — Minimal Style Guide

Palette
- Cream: #FFF8F2 (background)
- Blush: #F6D6D3 (soft card/backdrop)
- Burgundy: #6A0D2A (primary CTA)
- Rosewood: #8B3B46 (secondary CTA / headings)
- Latte: #E9DCC9 (muted surfaces)
- Slate: #475569 (body text / UI chrome)

Typography
- Headings: Playfair Display (serif)
- Body: Inter (or system sans)
- Head sizes:
  - H1: 48px / 56px line-height (hero)
  - H2: 32px
  - H3: 20-24px
  - body: 16px

Spacing & Radii
- Spacing scale: 4,8,12,16,24,32,40
- Border radius: 12-24px for main cards, 8px for buttons

UI pointers
- Use large imagery and audio play affordances.
- Cards should be airy: padding 24px, subtle shadows, rounded 16px

Tone & Copy
- Voice: warm, encouraging, intimate — speak directly to creators and readers
- Microcopy examples:
  - Recording CTA: "Capture your voice"
  - Workshop CTA: "Share this chapter — get heartfelt feedback"
  - Save: "Save draft"
